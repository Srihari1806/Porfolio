# Portfolio repo: how to structure

- `README.md` (landing)
- `docs/bio.md` (personal bio)
- `samples/` (all sample work)
- `assets/` (audio/images)
- `contact.md` (contact and links)

**Git commands to update:**

```bash
# copy files in place, then:
git add README.md docs/bio.md samples/* portfolio_structure.md
git commit -m "Update portfolio to Srihari Beesetti profile: lyrics, scripts, SEO samples"
git push origin main
```
